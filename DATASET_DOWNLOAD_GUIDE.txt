================================================================================
MineInsight Dataset Download Guide (arXiv:2506.04842)
================================================================================

QUICK SUMMARY:
- 38,000 RGB frames (Tracks 1-2 only)
- 53,000 VIS-SWIR frames (all tracks)
- 108,000 LWIR thermal frames (all tracks)
- 35 target classes (15 mines + 20 distractors)
- 6 sequences total (2 per track x 3 tracks)
- YOLOv8 bounding box annotations + SAM2 segmentation masks

MINIMUM DOWNLOAD (RGB + LWIR + Labels):
  Total: ~44 GB
  - RGB images (4 sequences): 34.4 GB
  - LWIR images (6 sequences): 9.3 GB
  - All labels: ~56 MB

FULL DOWNLOAD (all modalities + masks):
  Total: ~56 GB
  - Add VIS-SWIR: 11.7 GB
  - Add SAM2 masks: 115 MB

#!/bin/bash
mkdir -p /mnt/forge-data/datasets/mineinsight
cd /mnt/forge-data/datasets/mineinsight

# Check space first
echo "Free space:"
df -h /mnt/forge-data

# Download all tracks (RGB + LWIR only)
for url in \
  "https://mineinsight.short.gy/6Mvbjx:track_1_s1_rgb.zip" \
  "https://mineinsight.short.gy/Cy5OeN:track_1_s1_rgb_labels.zip" \
  "https://mineinsight.short.gy/LV5XZ1:track_1_s1_lwir.zip" \
  "https://mineinsight.short.gy/Z8fqcY:track_1_s1_lwir_labels.zip" \
  "https://mineinsight.short.gy/RBRY3I:track_1_s2_rgb.zip" \
  "https://mineinsight.short.gy/ncoHYs:track_1_s2_rgb_labels.zip" \
  "https://mineinsight.short.gy/PYfYwf:track_1_s2_lwir.zip" \
  "https://mineinsight.short.gy/bYpES1:track_1_s2_lwir_labels.zip" \
  "https://mineinsight.short.gy/OKXGyT:track_2_s1_rgb.zip" \
  "https://mineinsight.short.gy/yJ4vKD:track_2_s1_rgb_labels.zip" \
  "https://mineinsight.short.gy/Tkb2ra:track_2_s1_lwir.zip" \
  "https://mineinsight.short.gy/00s2Te:track_2_s1_lwir_labels.zip" \
  "https://mineinsight.short.gy/mZSLV8:track_2_s2_rgb.zip" \
  "https://mineinsight.short.gy/5ZEVE9:track_2_s2_rgb_labels.zip" \
  "https://mineinsight.short.gy/CuoFOX:track_2_s2_lwir.zip" \
  "https://mineinsight.short.gy/uvLeoo:track_2_s2_lwir_labels.zip" \
  "https://mineinsight.short.gy/UoD78c:track_3_s1_lwir.zip" \
  "https://mineinsight.short.gy/caHk6F:track_3_s1_lwir_labels.zip" \
  "https://mineinsight.short.gy/1XFhHc:track_3_s2_lwir.zip" \
  "https://mineinsight.short.gy/aT8QZL:track_3_s2_lwir_labels.zip"
do
  IFS=':' read -r link filename <<< "$url"
  echo "Downloading $filename..."
  wget -O "$filename" "$link"
done

echo "Extracting all files..."
unzip -q "*.zip"
rm *.zip

# Get calibration and targets
REPO="https://raw.githubusercontent.com/mariomlz99/MineInsight/main"
wget -O targets_list.yaml "$REPO/tracks_inventory/targets_list.yaml"

mkdir -p intrinsics_calibration extrinsics_calibration
wget -O intrinsics_calibration/rgb_camera_intrinsics.yaml \
  "$REPO/intrinsics_calibration/rgb_camera_intrinsics.yaml"
wget -O intrinsics_calibration/lwir_camera_intrinsics.yaml \
  "$REPO/intrinsics_calibration/lwir_camera_intrinsics.yaml"

echo "Complete! Dataset at /mnt/forge-data/datasets/mineinsight"

================================================================================
KEY INFORMATION FOR TRAINING
================================================================================

TARGETS MAPPING:
  - 35 total classes (IDs: 1-48)
  - Track 1: 23 targets (PFM-1 mines + household items)
  - Track 2: 20 targets (MON-90, MON-50 mines + items)
  - Track 3: 5 targets (C-3, M-35, PROM-1, VS-50 mines)
  - Get mapping from: targets_list.yaml

ANNOTATION FORMAT (YOLOv8):
  Each .txt file contains:
  <class_id> <x_center_normalized> <y_center_normalized> <width_normalized> <height_normalized>

IMPORTANT NOTES:
  - Track 3 has NO RGB images (camera failure) - LWIR only
  - LWIR labels exist in two variants:
    * Reprojected (recommended): geometrically consistent
    * Automatic: pipeline-generated, denser
  - SAM2 masks available for RGB/SWIR on Tracks 1-2 only
  - Dataset v2 (Feb 2026) with 437,000+ revised labels
  - All images are RGB/LWIR/SWIR .jpg format

EXPECTED DOWNLOAD TIME:
  - Minimal (44 GB) on 100 Mbps: ~2-3 hours
  - Full (56 GB) on 100 Mbps: ~3-4 hours
  - Largest single file: Track 1 Seq 2 RGB (12 GB) and Track 2 Seq 2 RGB (15.8 GB)
